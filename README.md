# CRUD Login PHP + Database

## Deskripsi
Program CRUD sederhana menggunakan PHP dan MySQL.
Fitur:
- Login user
- Menampilkan daftar mata kuliah dari database

## Teknologi
- PHP
- MySQL
- XAMPP

## Cara Menjalankan
1. Import database.sql ke phpMyAdmin
2. Simpan folder project ke htdocs
3. Jalankan Apache dan MySQL
4. Akses melalui browser
